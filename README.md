# 纸质书阅读器（Paper Book Reader）

一个**单文件、纯前端、离线可用**的电子书阅读器。无需服务器、无需安装、无外部依赖打包，双击 HTML 文件即可在浏览器中阅读 PDF / TXT / Markdown，并支持划词批注、整页笔记、进度保存与笔记导入导出。

## 文件说明

| 文件 | 说明 |
| --- | --- |
| `纸质书阅读器.html` | 中文界面版本 |
| `PaperBookReader.html` | 英文界面版本（功能与中文版完全一致） |

两个版本共用同一套本地存储键，**笔记与阅读进度可在中英文版本间共享**。

## 功能特性

- **多格式支持**：PDF、TXT、Markdown（`.md` / `.markdown`）
- **扫描版 PDF 支持**：无文本层的扫描版 PDF 会逐页渲染为图片，并提供「整页笔记」入口
- **划词笔记**：选中文字后右键 →「添加笔记」，自动高亮并插入笔记框
- **整页笔记**：扫描版 PDF 页面底部「Page Note / 整页笔记」按钮，记录整页心得
- **笔记面板**：右侧面板汇总当前书籍全部笔记，点击可跳转定位
- **笔记导入 / 导出**：以 JSON 文件备份或迁移笔记，相同位置自动合并
- **阅读进度**：自动记录并恢复滚动位置（每 5 秒及滚动时保存）
- **明暗主题**：一键切换亮 / 暗阅读模式
- **悬浮提示**：所有工具栏按钮与笔记操作均带 Tooltip 说明
- **大文件支持**：书籍原文存入 IndexedDB（突破 localStorage ~5MB 限制），仅元数据存 localStorage

## 使用方法

1. 用现代浏览器（Chrome / Edge / Firefox 等）打开 `纸质书阅读器.html` 或 `PaperBookReader.html`
2. 点击左上角「加载文件 / Load File」，或直接把文件拖入页面
3. 阅读时选中文字 → 右键 →「添加笔记 / Add Note」即可批注
4. 关闭页面后再次打开，会自动恢复上次阅读的书籍、进度与笔记

## 数据存储说明

所有数据仅保存在**本地浏览器**中，不会上传任何服务器：

| 数据 | 存储位置 |
| --- | --- |
| 书籍原文（PDF / 大文本） | IndexedDB（`paperBookReaderDB`） |
| 文件名、类型、大小、进度、笔记、主题 | `localStorage` |

> 如需彻底清理：在浏览器控制台执行
> ```js
> localStorage.clear();
> indexedDB.deleteDatabase('paperBookReaderDB');
> ```

## 注意事项

- 首次加载会从 CDN 引入 `pdf.js`、`marked`、`highlight.js`，**首次使用需联网**；之后浏览器通常会缓存这些脚本
- 笔记导出为 JSON 文件，可在另一台设备或浏览器中通过「导入笔记」恢复
- 本阅读器面向个人离线阅读与学习批注，不收集任何用户信息

## 技术栈

- 渲染：pdf.js（PDF）、marked + highlight.js（Markdown 高亮）
- 存储：IndexedDB + localStorage
- 无构建步骤，单 HTML 文件即可运行
